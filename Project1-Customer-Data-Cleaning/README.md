# Project 1 - Customer Data Cleaning

I cleaned a raw customer orders dataset of 1,200 rows and 14 columns using Microsoft Excel.

What I did:
- Removed 12 empty columns from the raw dataset
- Fixed inconsistent date formatting
- Found 309 blank values in CouponCode column and replaced with "No Coupon Code"
- Checked for duplicates — found 0 duplicates
- Converted dataset into Excel Table using Ctrl+T for easy filtering and sorting

Result: Dataset is fully cleaned and ready for analysis.

Tools Used: Microsoft Excel
