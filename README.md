# DecodeLabs Data Analytics Internship

This repository contains all three projects completed as part of my DecodeLabs Data Analytics Internship. I am a Google Certified Data Analyst with a BSc in Computer Science from Hazara University.

Projects:

1. Customer Data Cleaning (Microsoft Excel)
Cleaned a raw dataset of 1,200 rows. Found 309 missing CouponCode values, fixed date formatting, removed 12 empty columns, and confirmed 0 duplicates. Dataset fully prepared for analysis.

2. Exploratory Data Analysis (Google Sheets)
Analyzed 1,200 e-commerce orders. Total revenue was PKR 1,264,761, average order value PKR 1,054, Printer was most ordered product, Chair generated highest revenue, and 57 outliers were detected. Built a 4-chart dashboard.

3. E-Commerce Data Analysis using SQL (Google BigQuery)
Wrote 5 SQL queries to analyze 1,200 orders. Cancelled orders were highest at 20.8%, average order value was PKR 1,054, and Chair generated highest revenue despite Printer having most orders.

Tools Used: Microsoft Excel | Google Sheets | SQL | Google BigQuery
