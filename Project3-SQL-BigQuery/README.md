I analyzed an E-Commerce dataset of 1,200 orders using SQL queries in Google BigQuery.
What I did:

Analyzed product wise orders — Printer had highest orders
Calculated total revenue — PKR 1,264,761 and product wise revenue — Chair generated highest revenue PKR 195,620
Checked orders by status — Cancelled orders were highest at 20.8%
Calculated average order value — PKR 1,054 per order
Tracked monthly sales trend from Jan 2023 to 2025

SQL Concepts Used: SELECT, GROUP BY, ORDER BY, COUNT, SUM, AVG, FORMAT_DATE
Key Insight: Despite Printer being the most ordered product, Chair generated the highest revenue — proving that order volume does not always mean highest earnings.
Tools Used: Google BigQuery | SQL
